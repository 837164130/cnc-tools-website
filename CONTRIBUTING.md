# Contributing to CCCNC Website

## 开发流程

### 环境准备
1. 克隆仓库
```bash
git clone https://github.com/837164130/cnc-tools-website.git
cd cnc-tools-website
```

### 开发规范

#### HTML
- 使用语义化标签
- 确保响应式设计
- 添加适当的 ARIA 标签

#### CSS
- 使用 CSS 变量定义颜色和尺寸
- 遵循 BEM 命名规范
- 确保移动端优先

#### JavaScript
- 使用 ES6+ 语法
- 模块化开发
- 添加适当的注释

### 提交规范

```
<type>: <subject>

<body>

<footer>
```

类型：
- feat: 新功能
- fix: 修复
- docs: 文档
- style: 格式
- refactor: 重构
- test: 测试
- chore: 构建

### 分支管理

- main: 主分支
- develop: 开发分支
- feature/*: 功能分支
- hotfix/*: 热修复分支

### 代码审查

1. 提交前自检
2. 创建 Pull Request
3. 代码审查
4. 合并到主分支

## 联系方式

如有问题，请联系：
- Email: support@cccnc.com

# Changelog

## [1.0.0] - 2025-04-26

### Added
- 完整的产品数据库（7大分类，18个子分类，200+产品）
- 18个产品详情页面，包含规格表、价格、SVG图纸
- 产品搜索功能，支持按型号、名称、关键词搜索
- 产品对比功能，最多3款产品同时对比
- 切削参数计算器，根据材料和刀具计算最佳参数
- 多语言支持（中文、俄语、英语）
- 响应式设计，适配桌面、平板、手机
- PWA支持，可安装为桌面应用，离线访问
- 图片懒加载、代码分割、缓存策略
- SEO优化：sitemap.xml、robots.txt、语义化标签
- 打印优化：专门的打印样式表
- 主题切换：浅色/深色模式
- 键盘快捷键：/搜索、T主题、H首页、C联系
- 收藏功能：本地存储收藏的产品
- 分享功能：支持 WhatsApp、Telegram、邮件分享
- 面包屑导航
- 回到顶部按钮
- 页面加载器
- 通知系统
- 分析追踪
- Service Worker 离线支持
- 移动端导航菜单
- 滚动动画效果
- 产品推荐系统
- 快速查看功能
- 语言切换器
- 性能优化

### Changed
- 优化了产品页面的导航和布局
- 更新了首页的产品分类展示
- 改进了联系页面的表单功能
- 增强了产品详情页面的技术图纸

### Fixed
- 修复了移动端导航的显示问题
- 优化了图片加载性能
- 改进了打印样式
